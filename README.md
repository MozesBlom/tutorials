# Tutorials in evolutionary genomics

* [Computer practical Phylogenetics Course University of Potsdam 2023](https://github.com/MozesBlom/tutorials/tree/main/2023_Phy_Eco_Evol/README.md)<br>
A tutorial that explores the sources and consequences of coalescent variation across genomes using a coalescent simulator. This tutorial was developed as a computer practical for the Phylogenetics in Evolution and Ecology 2023 course taught by [Faysal Bibi](https://amniota.org/phylogenetics/) at the Museum für Naturkunde Berlin. It includes code snippets and examples from [Georgia Tsambos](https://github.com/gtsambos/2022-ts-workshops) and [Simon Martin](https://github.com/simonhmartin/tutorials/blob/master/topology_weighting/).
